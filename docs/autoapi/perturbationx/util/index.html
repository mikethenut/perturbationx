perturbationx.util.connect_adjacency_components(adj: numpy.ndarray, nodes=None, weights=(1.0,), seed=None)

Connect the components of a graph represented by an adjacency matrix.

Parameters:

  • adj (np.ndarray) – The adjacency matrix of the graph.

  • nodes (list, optional) – The nodes to connect. If specified, edges will only be added between these nodes (when possible).

  • weights (list, optional) – The weights to assign to the edges. Each edge will be assigned a random weight from this list.

  • seed (int, optional) – The seed for the random number generator.

Raises:

ValueError – If the adjacency matrix is not square.

perturbationx.util.get_neighborhood_components(graph: networkx.DiGraph, source_nodes: set, max_distance=0, neighborhood_type='union')

Get the neighborhood of a set of nodes in a graph.

Parameters:

  • graph (nx.DiGraph) – The graph to search for neighbors in.

  • source_nodes (set) – The nodes to find neighbors of.

  • max_distance (int, optional) – The maximum distance from leading nodes that neighbors can be. If 0, no neighbors will be included. If 1, only the neighbors of the nodes will be included. If 2, the neighbors of the neighbors of the nodes will be included, and so on. Defaults to 0.

  • neighborhood_type (str, optional) – The type of neighborhood to find. Can be one of “union” or “intersection”. If “union”, all nodes within the maximum distance from any leading node are returned. If “intersection”, only nodes within the maximum distance from all leading nodes are returned. Defaults to “union”.

Raises:

ValueError – If max_distance is less than 0 or neighborhood_type is not “union” or “intersection”.

Returns:

The nodes and edges in the neighborhood. They are returned as a pair of sets.

Return type:

(set, set)

perturbationx.util.get_shortest_path_components(graph: networkx.DiGraph, endpoints: set, directed=False, length_tolerance=0)

Find paths between nodes in a graph and return the nodes and edges in those paths.

Parameters:

  • graph (nx.DiGraph) – The graph to search for paths in.

  • endpoints (set) – The nodes to find paths between.

  • directed (bool, optional) – Whether to search for directed or undirected shortest paths.

  • length_tolerance (int | float, optional) – The tolerance for the length of the shortest paths. If 0, only the shortest paths are returned. If length_tolerance is an integer, it is interpreted as an absolute length. If length_tolerance is a float, it is interpreted as a percentage of the length of the shortest path. Defaults to 0.

Raises:

ValueError – If length_tolerance is not a number or is negative.

Returns:

The nodes and edges in the paths. They are returned as a pair of lists.

Return type:

(list, list)
