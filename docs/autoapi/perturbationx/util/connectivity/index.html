connect_adjacency_components(adj[, nodes, weights, seed])

Connect the components of a graph represented by an adjacency matrix.
perturbationx.util.connectivity.connect_adjacency_components(adj: numpy.ndarray, nodes=None, weights=(1.0,), seed=None)

Connect the components of a graph represented by an adjacency matrix.

Parameters:

  • adj (np.ndarray) – The adjacency matrix of the graph.

  • nodes (list, optional) – The nodes to connect. If specified, edges will only be added between these nodes (when possible).

  • weights (list, optional) – The weights to assign to the edges. Each edge will be assigned a random weight from this list.

  • seed (int, optional) – The seed for the random number generator.

Raises:

ValueError – If the adjacency matrix is not square.
