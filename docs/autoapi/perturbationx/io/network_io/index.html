perturbationx.io.network_io.read_dsv(filepath: str, default_edge_type='infer', delimiter='\t', header_cols=DEFAULT_DATA_COLS)

Read a DSV file and return a list of edges.

Parameters:

  • filepath (str) – Path to the DSV file.

  • default_edge_type (str, optional) – The default edge type to use if none is specified in the file. Defaults to “infer”.

  • delimiter (str, optional) – The delimiter used in the DSV file. Defaults to ” “.

  • header_cols (list, optional) – The column names of the columns in the DSV file. Defaults to [“subject”, “object”, “relation”, “type”].

Returns:

A list of edges.

Return type:

list

Validate a networkx.DiGraph object for use with toponpa.

Parameters:

  • graph (nx.DiGraph) – The graph to validate.

  • allowed_edge_types (list) – List of allowed edge types. Edges with types not in this list will be replaced with “infer”.

Raises:

perturbationx.io.network_io.to_edge_list(graph: networkx.DiGraph, edge_type='all', data_cols=DEFAULT_DATA_COLS)

Convert a networkx.DiGraph to a list of edges.

Parameters:

  • edge_type (list | str, optional) – List of edge types to include. If “all”, all edges will be included. Defaults to “all”.

  • data_cols (list, optional) – List of data columns to include. Defaults to [“subject”, “object”, “relation”, “type”].

Returns:

perturbationx.io.network_io.write_dsv(graph: networkx.DiGraph, filepath: str, edge_type='all', delimiter='\t', data_cols=DEFAULT_DATA_COLS, header=None)

Write a networkx.DiGraph to a delimited file.

Parameters:

  • graph (nx.DiGraph) – The graph to write.

  • filepath (str) – The path to write the file to.

  • edge_type (list | str, optional) – List of edge types to include. If “all”, all edges will be included. Defaults to “all”.

  • delimiter (str, optional) – The delimiter to use in the DSV file. Defaults to ” “.

  • data_cols (list, optional) – List of data columns to include. Columns not in [“subject”, “object”, “relation”, “type”] will be ignored. Defaults to [“subject”, “object”, “relation”, “type”].

  • header (list, optional) – List of header values to use. Must be given in the same order as data_cols. Defaults to None.

Raises:

ValueError – If the length of the header list does not match the length of the data_cols list.
