NPAResultBuilder

Class for building NPAResult objects. Node attributes can be only passed for nodes in the graph with type "core".
class perturbationx.result.NPAResultBuilder.NPAResultBuilder(graph: networkx.DiGraph, datasets: list)

Class for building NPAResult objects. Node attributes can be only passed for nodes in the graph with type “core”. Each core node should have an attribute “idx” that is a unique integer index for that node. Node attributes should be passed as a list of values that is ordered by the node indices.

Parameters:

  • graph (networkx.DiGraph) – The graph used to generate the result.

  • datasets (list) – The datasets used to generate the result.
