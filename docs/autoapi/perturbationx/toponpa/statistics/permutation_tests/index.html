Test the null hypothesis that the ordering of boundary coefficients does not affect the perturbation score.

Parameters:

  • lap_b (np.ndarray | sp.sparray) – The Lb boundary Laplacian.

  • lap_c (np.ndarray | sp.sparray) – The Lc core Laplacian.

  • lap_q (np.ndarray | sp.sparray) – The Q core Laplacian.

  • boundary_coefficients (np.ndarray) – The boundary coefficients.

  • core_edge_count (int) – The number of edges in the core network.

  • permutation_rate (float, optional) – The fraction of boundary coefficients to permute. Defaults to 1.

  • iterations (int, optional) – The number of permutations to perform. Defaults to 500.

  • seed (int, optional) – The seed for the random number generator. Defaults to None.

Returns:

The distribution of perturbation scores under the null hypothesis.

Return type:

list

perturbationx.toponpa.statistics.permutation_tests.test_core_permutations(adj_perms: dict, boundary_coefficients: numpy.ndarray, lap_b: numpy.ndarray | scipy.sparse.sparray, core_edge_count: int, exact_boundary_outdegree=True, lap_q=None, full_permutations=True)

Test the null hypothesis that the distribution of core edges does not affect the perturbation score.

Parameters:

  • adj_perms (dict) – The adjacency matrices of the core network permutations. The keys are the permutation names, the values are the adjacency matrices. The adjacency matrices may be sparse or dense.

  • boundary_coefficients (np.ndarray) – The boundary coefficients.

  • lap_b (np.ndarray | sp.sparray) – The Lb boundary Laplacian.

  • core_edge_count (int) – The number of edges in the core network.

  • exact_boundary_outdegree (bool, optional) – Whether to use the exact boundary outdegree. If False, the boundary outdegree is set to 1 for all core nodes with boundary edges. Defaults to True.

  • lap_q (np.ndarray | sp.sparray, optional) – The Q core Laplacian. Required if full_permutations is False. Defaults to None.

  • full_permutations (bool, optional) – Whether to use the full permutation matrix for each core permutation. Partial permutations sample core coefficients, while full permutations sample perturbation scores. Defaults to True.

Raises:

ValueError – If full_permutations is False and lap_q is None.

Returns:

The distribution of perturbation scores under the null hypothesis.

Return type:

list
